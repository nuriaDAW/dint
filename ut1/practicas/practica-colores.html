Práctica: Recopilación de herramientas de gestión de color

Realiza una pagina web que contenga todos los enlaces a las distintas herramientas de gestion de color que indica en el apartado color del tema. Investiga cada una de ellas. En esta web utiliza tus colores de marca.

Teniendo en cuenta el siguiente articulo https://creativolandia.com/como-elegir-los-colores-para-mi-web/. Recopila información sobre la psicologia de los colores.

Psicología de los colores

Conocer el significado que puede tener cada color en la mente humana puede ser muy interesante para elegir la mejor paleta de colores. Además, este conocimiento permite escoger una gama de colores que acompañe y ayude a transmitir de forma fiel y correcta los atributos de la marca.

Blanco

El color blanco es uno de los más usados en el diseño web actual, ya que transmite al usuario una sensación de pureza, frescura, limpieza y simplicidad. Además, usar este color como fondo permite que el ojo del usuario descanse y perciba mejor las llamadas de acción.

Rojo

El rojo es un color que transmite a las personas sensaciones ligadas a la pasión, la impulsividad, el deseo y la fuerza. Por lo tanto, es muy útil para webs que vendan productos o traten temas relacionados con el amor o el deseo.

Marron

Es el color menos atractivo y se usa más que otra cosa como color neutro o de paso. Además de a cosas agradables como el chocolate, la tierra o los troncos de los árboles puede tener evocaciones escatológicas.

Naranja

El color naranja está ligado a sensaciones de calidez, creatividad, éxito y ánimo. Esto hace que sea un color muy interesante para crear webs personales o de negocios.

Amarillo

El amarillo suele estar relacionado con significados como energía, diversión, alegría e innovación. Sin embargo, este es el color que más contradicciones genera entre los usuarios ya que puede llegar a generar una sensación de irritación.

Verde

El color verde suele estar ligado a sentimientos relacionados con la naturaleza, el equilibrio, el crecimiento o la estabilidad. Es muy interesante su uso en una web porque ayuda a generar confianza en el usuario y que se sienta relajado.

Azul

El color azul suele ser un color muy utilizado a la hora de crear páginas web de negocios, ya que transmite sentimientos como libertad, fidelidad, progreso, seriedad y lealtad.

Morado

El color morado puede ser muy interesante a la hora de crear una web ya que transmite sensaciones ligadas con la serenidad, la elegancia, la sensualidad y el eclecticismo.

Rosa

El rosa puede proporcionar sensaciones muy buenas para los usuarios en una web, ya que va ligado a sentimientos de dulzura, delicadeza y amistad.

Gris

El gris es uno de los colores que menos sensaciones tiene ligadas, aunque se asocia con la paz y la tenacidad. Dependiendo de la cultura puede ser relacionado con la mediocridad o la indeterminación.

Negro

El color negro es uno de los más usados a la hora de crear páginas web, ya que proporciona sensaciones de poder, sobriedad, formalidad e, incluso, misterio.

Herramientas de gestión de color

Paletton

Generador de paletas de colores muy útil cuando hay un color base definido para realizar combinaciones de hasta cuatro colores.

Da la opción de seleccionar la combinación de colores con distintos niveles de saturación y brillo prediseñados a partir del esquema.

Sus dos puntos más fuertes son:

  • Distintas visualización de mapa de colores, que ayuda a visualizar la combinación de la gama de colores seleccionada.
  • Tablas de color y exportación en distintos formatos (html, css, less, sass, xml y text).

La mayor pega de la herramienta es que únicamente permite introducir un color en hexadecimal como base. La modificación posterior de los colores es mediante el ratón.

Material Design

Herramientas para selección de colores. Con la herramienta COLOR TOOL se pueden crear y compartir paletas de colores para la interfaz de usuario. Permite seleccionar un color primario, uno secundario y medir el nivel de accesibilidad de la combinación del color de la fuente y el fondo.

Además del nivel de accesibilidad, ofrece sugerencias sobre la aplicación de los colores a la página en distintos elementos, autogenerando incluso el código tanto del color como de los elementos con la opción "EXPORT".

Su mayor pega es que únicamente permite seleccionar un color primario y uno secundario.

Colour Lovers

Herramientas para crear paletas de colores seleccionando hasta cinco colores de forma libre e independiente.

Resulta más limitado como selector de colores para aplicaciones web, no ofrece muestras de cómo quedaría el esquema ni códigos web.

Algunas características requieren una cuenta.

  • Basic

    •

    Herramienta básica. No sugiere combinaciones de colores.

  • Copaso

    Herramienta avanzada. Como característica destacable con respecto a la herramienta básica son las combinaciones de colores en función a las teorías de color.

  • Photocopa

    Extrae los colores de una de las imágenes de su galería o una url (requiere autenticación) para usarlos en una paleta de colores.

Color Pix

Herramienta gratuita descontinuada.

Aunque ofrece paletas prediseñadas para descarga en línea, es una aplicación de escritorio portable.

La aplicación es un pequeño selector de colores que toma el píxel bajo el ratón de cualquier parte visible en el escritorio y lo transforma en una serie de formatos de color diferentes.

Tiene una serie de herramientas web asociadas (ColorSchemer) para obtener códigos de colores.

Color Wheel Pro

Software de pago algo desfasado. Ofrece una versión de prueba de 30 días, que no incluye la previsualización de webs prediseñadas con los colores escogidos.

Color Wheel Pro es una herramienta de software que ayuda a crear esquemas de color basados en la teoría del color.

No tiene ninguna ventaja con respecto a herramientas en línea como Paletton, salvo la posibilidad de elegir todos los colores directamente por su valor hexadecimal o rgb y el no depender de un recurso externo ni una conexión.

Adobe Color

Conjunto de herramientas para crear paletas, explorar paletas a partir de una galería de imágenes o extraer paletas o degradados de imágenes.

Permite formar paletas de hasta cinco colores creadas en base a teorías de color o de forma libre.

Con una cuenta de Adobe se pueden guardar y gestionar las paletas creadas en una biblioteca.

Incluye una herramienta de accesibilidad para daltónicos que comprueba si los colores seleccionados son aptos para ellos.

Al contrario que las herramientas diseñadas con orientación a web, no tiene previsualización de elementos web ni generación de código.

Material Design Palette

Generador de esquemas de color en base a dos colores seleccionados entre los disponibles.

Propone al usuario qué colores emplear de acuerdo con la paleta de color previamente seleccionada.

El esquema de colores se puede descargar en formato css, sass, less, svg, xml, png y polymer.

Color Combos

Sitio destinado a ayudar a los desarrolladores web a seleccionar y probar rápidamente combinaciones de colores para diseño web.

El corazón del sitio es el Combo Tester, que permite ver cómo diferentes combinaciones de hasta treinta colores funcionan juntas en la pantalla.

Una funcionalidad interesante es el extractor de colores de un sitio web mediante la url. Captura los valores hexadecimales de los colores y los añade a la paleta del Combo Tester.

Otra herramienta útil es el Font Color Tester que facilita la previsualización de una combinación de dos colores para fuentes.
