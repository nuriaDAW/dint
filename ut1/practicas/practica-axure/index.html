Práctica: Prototipos con Axure

Axure es una herramienta de prototipado muy utilizada en las empresas. Permite crear prototipos personalizados para cualquier dispositivo.

Además de imágenes se pueden añadir vídeos de forma nativa y audio mediante código javascript.

Provee de un catálogo de acciones bastante amplio y permite generar código a partir del prototipo creado.
