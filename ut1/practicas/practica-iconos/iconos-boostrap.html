Iconos en SVG

Incrusta iconos en formato svg en el html (a diferencia de un archivo de imagen externo). Se puede personalizar el estilo directamente en la etiqueta.

Iconos preformateados

Iconos de Glyphicon Halflings. Toman el estilo de <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">. El css se puede modificar utilizando uno propio.
