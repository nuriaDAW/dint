Se puede añadir (en <head>):

Ventajas de usar un kit

  • Rápida carga y conversión de los iconos.
  • Independencia con las versiones.

Inconvenientes de usar un kit

  • Requiere una cuenta de usuario para generar el código, que se asociará al usuario.
  • El cliente puede tener desactivado o bloqueado javascript.
