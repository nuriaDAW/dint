Herramientas de Planificación de Interfaces

MATERIAL DESIGN. COLOR TOOL

Herramienta para crear y compartir paletas de colores para la interfaz de usuario. Permite seleccionar un color primario, uno secundario y medir el nivel de accesibilidad de cualquier combinación de los colores de la paleta.

PALETTON

Generador de paletas de colores muy útil cuando hay un color base definido para realizar combinaciones de hasta cuatro colores.

Provee tablas de color y exportación en distintos formatos (html, css, less, sass, xml y text).

HTML color codes

Herramienta de selección de códigos de color html, códigos de color hexadecimales, valores RGB y HSL, carta de colores y nombres de colores HTML.

Color Contrast Checker

Herramienta sencilla desarrollada por Mari Johannessen para ayudar a los desarrolladores y diseñadores a asegurarse de que sus paletas de colores son accesibles en la web. Basado en los estándares WCAG 2.0 Nivel AA.

Más herramientas de esquemas de color

Google Fonts

Selector de fuentes de Google. Genera un enlace a los estilos de fuente de la API de Google y el css asociado.

Google Icons

Selector de iconos de Google. Genera un enlace a los estilos de icono de la API de Google.

También permite la descarga de los iconos en png y svg.

El set básico está ampliado con contribuciones de los usuarios en: Material Design Icons

Balsamiq Wireframes

Herramienta para la creación de prototipos o maquetas. Es similar en cuanto a funcionalidad a los editores de imagen, permite arrastrar, soltar, guardar mockups en un archivo, copiar y pegar, deshacer, etc.

Incluye una amplia variedad de objetos prediseñados listos para usar y personalizables (hasta cierto punto), como: barras de estado, menús, barras de progreso, etc.

Además, permite exportar el diseño que realizes en png, pdf e incluso al portapapeles.
